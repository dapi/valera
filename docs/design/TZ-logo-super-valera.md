## 1. О проекте

**Супер Валера** — это AI-powered Telegram бот для автосервисов, специализирующийся на кузовном ремонте и покраске. Бот ведет естественный диалог с клиентами, анализирует фото повреждений и помогает записаться на ремонт.

### Ключевые особенности
- **AI-first подход:** Бот общается как живой эксперт, без меню и кнопок
- **Визуальный анализ:** Клиенты присылают фото повреждений для оценки
- **Профессиональная экспертиза:** 10+ лет опыта в терминологии кузовного ремонта
- **Дружелюбность:** Помощник, а не робот

### Целевая аудитория продукта
- Владельцы автомобилей с повреждениями кузова (после ДТП)
- Клиенты со страховыми случаями (ОСАГО/КАСКО)
- Клиенты покраски и полировки

### Целевая аудитория проекта (кто будет использовать логотип)
- Владельцы автосервисов (устанавливают бота себе)
- B2B маркетинг для SaaS-продукта

## 2. Требования к логотипу

### 2.1 Концепция

**Название:** Супер Валера

**Позиционирование:**
- Современный AI-помощник
- Профессионал кузовного ремонта
- Дружелюбный и доступный
- Технологичный, но не холодный

**Ассоциации (что должен вызывать логотип):**
- Технологичность (AI, автоматизация)
- Профессионализм (авто, ремонт, мастерство)
- Дружелюбность (помощник, не корпоративный монстр)
- Надежность (качество, доверие)

**Антиассоциации (чего избегать):**
- Устаревшие технологии (гаечные ключи, старые авто)
- Агрессивность или холодность
- Детский/несерьезный стиль
- Чрезмерная корпоративность

### 2.2 Стилистика

**Стиль:** Современный, минималистичный, технологичный

**Направление:**
- Flat design или minimal 3D
- Чистые линии
- Современные пропорции
- Возможна геометрическая стилизация

**Референсы по стилю (для вдохновения):**
- Telegram бренд (современность, доступность)
- Notion (минимализм + дружелюбность)
- Anthropic Claude (AI с человечностью)
- Intercom (B2B SaaS + дружелюбность)

### 2.4 Цветовая палитра

**Основные цвета:**
| Цвет | Ассоциация | Рекомендация |
|------|------------|--------------|
| Синий | Технологии, доверие, профессионализм | Основной или акцентный |
| Оранжевый/Желтый | Автосервис, энергия, дружелюбность | Акцентный |
| Темно-серый | Авто, премиальность | Для текста/деталей |

**Альтернативные варианты:**
- Градиент синий → фиолетовый (AI/tech)
- Минтовый/бирюзовый (свежесть, современность)

**Избегать:**
- Красный (агрессия, опасность)
- Коричневый (устаревший стиль)
- Неоновые кислотные цвета

### 2.5 Элементы логотипа

**Обязательные компоненты:**
1. **Иконка/знак** — для использования как аватар (Telegram, соцсети)
2. **Логотип с текстом** — для маркетинга и документации

**Возможные визуальные метафоры:**

| Метафора | Описание |
|----------|----------|
| Чат-бабл + авто | Диалог об автомобилях |
| AI + кузов | Интеллектуальный анализ повреждений |
| Буква "В" стилизованная | Персонификация Валеры |
| Робот-помощник | Дружелюбный AI-ассистент |

**Интересная идея:**
Персонаж "Супер Валера" — стилизованный мастер/помощник с AI-элементами. Не обязательно человек, может быть абстрактный персонаж.

## 3. Технические требования

### 3.1 Форматы

**Векторные (обязательно):**
- AI (Adobe Illustrator) — исходник
- SVG — для веб
- PDF — для печати

**Растровые:**
- PNG с прозрачностью (512x512, 1024x1024, 2048x2048)
- ICO для фавиконки (16x16, 32x32, 48x48)

### 3.2 Цветовые версии

| Версия | Описание | Применение |
|--------|----------|------------|
| Полноцветная | Основная версия | Веб, маркетинг |
| Монохромная (черная) | Для документов | Печать на светлом |
| Инвертированная (белая) | Для темного фона | Темные темы, фото |
| Черно-белая | Минимальная | Факс, штамп |

### 3.3 Размеры и масштабирование

**Минимальный размер:**
- Иконка: 16x16 px (должна быть узнаваема)
- Логотип с текстом: 80px по ширине

**Области применения:**
- Аватар Telegram бота (круглый формат!)
- Веб-сайт (хедер, футер)
- Документация (README, docs)
- Маркетинговые материалы
- Favicon
- App icon (если будет приложение)

### 3.4 Safe zone

Определить минимальные отступы вокруг логотипа (обычно 1/4 высоты логотипа)

### 4.1 Telegram-специфика

**Критически важно:** Логотип будет использоваться как аватар Telegram бота!

- Узнаваемость в малом размере (40px)

- Название на русском языке
- Понятно русскоязычной аудитории
- Отражает дружелюбность (имя "Супер Валера" — простое, народное)

- Профессиональный вид для презентаций владельцам автосервисов

### 10.5 Советы по работе с Nano Banana для логотипов

**Обязательно указывать:**

- "isolated white background" — для чистого фона

- Точные цвета в HEX
- Аспект 1:1 для аватаров

**Итеративный процесс:**
1. Сгенерировать лист из 9 вариантов
2. Выбрать лучшее направление
3. Сгенерировать 6 вариаций выбранного
4. Доработать в векторном редакторе (Figma/Illustrator)

- Кириллица может генерироваться с ошибками — лучше добавлять текст отдельно

### 11.5 Настройки Ideogram

**Рекомендуемые параметры:**
- **Style:** Auto или Design (не Realistic)
- **Aspect Ratio:** 1:1 для иконок, 16:9 для горизонтальных
- **Magic Prompt:** включить для улучшения
- **Negative prompt:** blurry, distorted text, ugly, amateur

**Советы для кириллицы:**
- Писать текст в кавычках: "Супер Валера"
- Указывать "Russian text" или "Cyrillic"
- Проверять результат — иногда буквы искажаются
- Лучше генерировать 4-8 вариантов и выбирать лучший
